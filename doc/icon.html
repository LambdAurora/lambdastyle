Icons

Icons are a great way to communicate things without text. They are integrated inside many components.

Basic icons

Icons are declared with the ls-icon css class in an i tag. But there is a lot of custom icons that you can include in your page.

Don't forget to include the aria-hidden="true" data for screen readers.

Feather icons

One of the possible icon set which can be included is the feather icons set.

The icons are specified with the feather-data data.

Icon sizes

You can resize the icons by changing the font-size css style.

There is also some helpers with the ls-size data.
