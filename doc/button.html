Buttons

Buttons communicate an action a user can take. They are placed throughout your user interface, in places like dialogs, forms, cards, and toolbars.

Basic button

Buttons are declared with the ls-btn css class. 

A basic button

Button variants

Buttons have different variants: flat, outlined, contained, icon, etc...

Variants are specified with the ls-variant data.

Button sizes

Buttons may have different sizes: small, medium or large. By default buttons are medium sized.

The size is modified with the ls-size data. Valid values are: small, medium and large.

Button states

Buttons are by default enabled, but they can also be disabled.

To disable a button, just add the disabled data.
